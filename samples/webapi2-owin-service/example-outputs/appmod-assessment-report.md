## Issues — Detail

### 🔴 Mandatory

#### 1. WebAPI 2 (`System.Web.Http`) → ASP.NET Core controllers

- **Files:** `OrdersController.cs:8, 12, 19, 47, 73`,
  `WebApiConfig.cs:1, 7, 13`
- **Why:** `System.Web.Http` is .NET Framework only. ASP.NET Core uses
  `Microsoft.AspNetCore.Mvc` with `ControllerBase` and `IActionResult`.
- **Recommendation:** Scenario — **".NET version upgrade"**.

#### 2. OWIN hosting → Kestrel + ASP.NET Core middleware

- **Files:** `Startup.cs:1–14`, `Web.config:5` (TFM)
- **Why:** `Microsoft.Owin` and `[OwinStartup]` are Framework-only. ASP.NET
  Core uses `Program.cs` minimal host.
- **Recommendation:** Same scenario as #1 — handled together.

#### 3. SQL injection in `List`

- **File:** `OrdersController.cs:53` (`customer` concatenated into `LIKE`)
- **Why:** Live vulnerability. The `LIKE '%...%'` pattern is especially
  dangerous because `%` and `_` allow data exfiltration.
- **Recommendation:** Parameterize — `WHERE CustomerName LIKE @c`,
  `cmd.Parameters.AddWithValue("@c", "%" + customer + "%")`.

#### 4. Replace `System.Data.SqlClient`

- **Files:** `OrdersController.cs:4, 22, 50`, `Web.config:5`
- **Recommendation:** Predefined task — **"Migrate to Managed Identity based
  Database on Azure"**.

#### 5. Replace `Integrated Security=True` connection string

- **File:** `Web.config:4`
- **Recommendation:** Same task as #4.

#### 6. Move `DownstreamApiKey` to Azure Key Vault

- **File:** `Web.config:9`
- **Recommendation:** Predefined task — **"Migrate to secured credentials by
  using Managed Identity and Azure Key Vault"**.

#### 7. Replace `System.Diagnostics.EventLog`

- **File:** `OrdersController.cs:84–86`
- **Recommendation:** Predefined task — **"Migrate to OpenTelemetry on
  Azure"**.

#### 8. Upgrade from .NET Framework 4.6.2

- **File:** `OrderApi.csproj:11`
- **Recommendation:** Scenario — **".NET version upgrade"** (.NET 8+).

### 🟠 Potential

#### 9. Replace `HttpWebRequest` with `HttpClient`

- **File:** `OrdersController.cs:75–82`
- **Why:** `HttpWebRequest` is obsolete; no DI, no retry, no socket pooling.
- **Recommendation:** `IHttpClientFactory` + Polly.

#### 10. `WebApiConfig.Register` has no global error filter

- **File:** `WebApiConfig.cs:14` (comment notes the gap)
- **Why:** Unhandled exceptions surface as raw stack traces. ASP.NET Core
  uses `app.UseExceptionHandler("/error")` + ProblemDetails.

#### 11. Internal-only downstream URL `http://internal-fulfillment.contoso.local`

- **File:** `Web.config:8`
- **Why:** Not reachable from Azure compute. Plan VNet integration or
  Private Endpoint or replace with Azure-hosted equivalent.
